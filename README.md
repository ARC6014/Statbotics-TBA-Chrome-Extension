THIS IS A FORK TO PORT FIREFOX (see OG here: https://github.com/ARC6014/Statbotics-TBA-Chrome-Extension)\
This extension overlays Statbotics data over TBA\
\
Key Features:\
\
    Team Stats: View a team’s EPA (it is basically a dynamic average of a team’s performance), percentile rankings, and season records directly on their team page.\
\
    Event Comparison: Access the mean and max EPA of teams at a specific event to gauge its difficulty level.\
\
    Rankings & Awards: On the rankings page, you can easily see which teams won awards alongside their average EPA in a total sense.\
\
    Match Predictions: See win-probability predictions directly on the match page before the game even begins, and see if the predictions were true.\
